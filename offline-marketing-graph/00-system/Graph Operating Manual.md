# 그래프 운영 원칙

## 목적

오프라인 마케팅 데이터를 실행 기록이 아니라 재사용 가능한 의사결정 자산으로 쌓습니다.

## 기본 단위

- 캠페인: 예산, 목표, 타깃, 기간이 있는 실행 묶음
- 장소: 팝업, 매장, 헬스장, 행사장, 상권 등 물리적 접점
- 파트너: 제휴처, 운영 대행사, 인플루언서, 커뮤니티, 입점처
- 소재: 배너, 전단, 쿠폰, QR, 시식대, POP, 제안서
- 실행 이벤트: 특정 날짜와 장소에서 일어난 실제 활동
- 성과 스냅샷: 특정 기준일/기간의 측정값
- 인사이트: 반복 가능한 배움, 가설, 의사결정 근거

## 생성 규칙

1. 새 실행은 먼저 캠페인 또는 실험 노트로 만든다.
2. 장소, 파트너, 소재가 재사용될 가능성이 있으면 반드시 독립 노트로 만든다.
3. 성과 수치에는 기간, 기준, 원천 데이터를 남긴다.
4. 링크가 3개 이상 생긴 실행 교훈은 인사이트 노트로 승격한다.
5. 완료된 캠페인은 결과 요약과 다음 액션을 남긴 뒤 `status: completed`로 바꾼다.

## 권장 네이밍

- 캠페인: `YYYY-MM 캠페인명`
- 장소: `장소명 - 지역`
- 파트너: `파트너명`
- 소재: `YYYY-MM 소재명`
- 성과 스냅샷: `YYYY-MM-DD 캠페인명 성과`
- 인사이트: `인사이트 - 핵심 문장`
